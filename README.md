# 🧪 Experimento Bootstrap

> **Nota de Portfólio:** Este projeto foi desenvolvido com o propósito exclusivo de praticar e demonstrar habilidades práticas com o **Bootstrap 5**, aplicando o framework para transformar uma estrutura simples de HTML em um layout responsivo e moderno.

---

## 🎯 Objetivo do Projeto

O foco principal foi migrar uma estrutura de página de produto/curso (contendo cabeçalho, imagem, bloco de informações e rodapé) para o ecossistema do Bootstrap. O laboratório serviu para entender como substituir o CSS manual por classes utilitárias e componentes nativos, acelerando o tempo de desenvolvimento.

---

## 🛠️ Recursos do Bootstrap Implementados

Com base na estrutura do código deste repositório, os seguintes conceitos do Bootstrap foram explorados:

* **Containers e Grid System (`.container`, `.row`, `.col-*`):** Utilizados para alinhar a seção de `imagem` e o bloco de `informacoes` lado a lado em telas grandes, e empilhá-los automaticamente em dispositivos móveis.
* **Componentes de Botões (`.btn`, `.btn-primary`):** Substituição dos botões nativos do HTML por botões estilizados, aplicando variações de cores, espaçamentos e efeitos de *hover* nativos do framework.
* **Tipografia e Espaçamento (`.mb-*`, `.py-*`, `.fw-bold`):** Uso de classes utilitárias para controlar as margens dos parágrafos, peso das fontes (como os pré-requisitos e horários) e preenchimento do cabeçalho e rodapé, eliminando a necessidade de regras
*  complexas no arquivo CSS externo.

---

## 💻 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo da página.
* **Bootstrap 5:** Framework principal utilizado para estilização rápida, responsividade e componentes visuais.
* **CSS3:** Utilizado apenas para ajustes finos e pontuais onde o framework não cobria a necessidade do design.
